# Play With Friends — Project Reference

## Stack
- **Framework:** Angular 19 (standalone components, signals)
- **Rendering:** SSG (static prerender via `@angular/ssr`) + CSR fallback
- **Search:** Lunr.js full-text index (built at runtime in `search.service.ts`)
- **Build:** `npm run build` → `ng build && node scripts/generate-sitemap.mjs`
- **Deploy:** GitHub Actions → GitHub Pages (`dist/playwithfriends/browser`)
- **Live URL:** https://playwithfriends.link/

## Component Notes

### NavComponent (`components/nav/`)
- Sticky, `z-index: 100`, `backdrop-filter: blur(8px)`
- Logo left, links right
- Active link: `--accent` color + `--accent-light` bg

### FilterBarComponent (`components/filter-bar/`)
- Emits `filtersChange` with `GameFilters` object
- Chips for: duration (short/medium/long), platform, price (free/cheap/medium/expensive), mode (local/remote)
- Player count: free-form number input
- "Clear filters" button appears only when `hasActiveFilters`

### GameCardComponent (`components/game-card/`)
- All games have `image: null` → shows first-letter placeholder with accent color
- Card lifts on hover (`translateY(-3px)`) + accent border
- Meta: Players, Duration, Price (color-coded via `.price-*` classes)

### HomeComponent (`pages/home/`)
- Hero: full-viewport, centered, gradient title (text → accent)
- Contains `<app-filter-bar>` + "Explore Games" button → navigates to `/games` with query params

### GamesComponent (`pages/games/`)
- 280px sticky sidebar + fluid grid (`auto-fill minmax(240px)`)
- Reads query params on init, syncs URL on filter change

### GameDetailComponent (`pages/game-detail/`)
- 360px image column (sticky) + prose column
- Sets `<title>`, meta description, og:* tags
- Injects JSON-LD `SoftwareApplication` schema

## Game Data Schema
```typescript
interface Game {
  slug: string;
  title: string;
  description: string;
  url: string;
  players: { min: number; max: number; softLimit: boolean };
  price: number;          // 0 = free
  duration: ('short' | 'medium' | 'long')[];
  platforms: ('web' | 'windows' | 'linux' | 'macos')[];
  multiplayerType: ('local' | 'remote')[];
  tags: string[];
  controls: ('smartphone-controller')[];
  image: string | null;   // all currently null → letter placeholder
}
```

## Old Svelte Site (reference for style origins)
The original Svelte site at playwithfriends.link used:
- **Font:** Open Sans (Google Font)
- **Background:** Pure `#000000` black
- **Text:** White (`#fff`)
- **Nav:** Right-aligned links only (no logo), underline on active
- **Filter bar bg:** `bg-cyan-900` → `#164e63`
- **Buttons:** `border-2 bg-transparent text-white font-semibold uppercase`
- **Decorative bg:** Fixed SVG (`mainboard.svg`) at 70vw/70vh, opacity 0.2, z-index -1
- The Angular rewrite intentionally modernised this with a proper design system.
